# Login-App with Auth


# Quick Install
1. Rename .env.example to .env
2. Update 
	DB_CONNECTION=mysql
	DB_HOST=127.0.0.1
	DB_PORT=3306
	DB_DATABASE=homestead
	DB_USERNAME=homestead
	DB_PASSWORD=secret and Social IDs in .env file to yours.
3. In command line : 
   - composer update
   - Php artisan key:generate
   - php artisan migrate
4. npm install
5. npm run dev
